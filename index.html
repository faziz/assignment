Assignment

A sample appliaction to show off REST web application using JEE6, JAXB, Jackson and simple JQuery.

Download this project as a .zip file Download this project as a tar.gz file

Welcome to RESTfull sample web application.

The idea is basically to come up with a general framework where any developer can drop in custom annotated Java classes and those be available as RESTified web services, supporting both XML and JSON serialization.

Ideally I would have rather used JEE6 with JAX-RS be done with everything, but due to some restrictions on available technologies to be used, I was not to use JAX-RS.

The project is built using Apache Maven, JDK 6, JPA 2, Glassfish V3 with JEE6 but with out JAX-RS because it needed to provide custom annotation support. Netbeans IDE was used to help in develop and deploy the application, but there is dependency on the IDE. Maven build is completely agnostic to any application server.

The application workflow is very simple.

org.faziz.assignment.web.LoadMetaDataListener is the application context listener that scans the classes, at the application startup, in org.faziz.assignment.service package and look for custom annotation @export, caches all the meta data in application context.

org.faziz.assignment.web.RestifiedServlet is used as the REST request processing servlet. It dispatches the request to appropriate request handler by looking at 1. HTTP method used (PUT, GET, POST, DELETE) and, 2. Request URI. If appropriate handler is found it executes the requests, and return the data in either XML or JSON format, depending on the the request Content-Type used.

On the web UI side is a simple HTML page with JQuery providing interactivity with the page and the server.

How to deploy.

If you are using Netbeans 7.x with GF V3, you just need to register the server with the project after that you should be able to Run-Debug the application from the IDE. If you don't have the IDE you can build from provided Maven build using the usual maven install command from the command prompt and manually deploy the application.

This application can only deploy on JEE6 web profile compliant web servers, since I am CDI and annotated context listener, however with little tweaks you should be able to run it on any web container.
